# colorTree 帮助手册

## 命令概述
`colorTree` 是一个可视化显示目录结构的命令行工具，支持彩色输出和多种自定义选项。

## 编译&运行
``` powershell
# 编译源码到 bin/
javac -d bin -cp "lib/*" src/com/ysc/test/Main.java
# 打包 jar
jar cfm colorTree.jar MANIFEST.MF -C bin .
# 运行 jar
java -jar .\colorTree.jar
```

## 使用语法
```cmd
colorTree [选项]
```

> **注意**：运行程序前请确保命令行使用 UTF-8 编码，可执行：
> 
> ```cmd
> chcp 65001
> ```
> 
> 否则输出可能异常。

## 选项表

| 参数             | 别名        | 描述                                                         |
| :--------------- | ----------- | :----------------------------------------------------------- |
| `-p <路径>`      | `--Path`    | 指定要展示的目录路径<br>支持 `%变量%` 或 `$env:变量` 格式的环境变量<br>不使用此选项时默认为当前路径 |
| `-od`            | `--OnlyDir` | 仅显示文件夹信息，不显示文件                                 |
| `-l <层数>`      | `--Layer`   | 设置最大显示层级深度（忽略负数值）                           |
| `-s <索引>`      | `--Style`   | 选择配色方案索引（有效值：0-8，默认0）                       |
| `-of <起始索引>` | `--Offset`  | 设置配色方案起始颜色索引（>6 的值会取余7）                   |
| `-a <列数>`      | `--Align`   | 设置文件大小对齐列位置（默认80）                             |
| `-k`             | `--KB`      | 使用1000进制单位（KB/MB/GB）<br>默认使用1024进制（KiB/MiB/GiB） |
| `-fk`            | `--ForceKB` | 强制使用KB单位格式<br>即使用1024进制也显示为KB/MB/GB         |
| `-r <文件路径>` | `--Redirect` | 执行重定向输出的路径<br>不会使用任何颜色方案 |
| `-v`             | `--Version` | 显示版本和版权信息                                           |
| `-h` | `--Help` | 显示帮助信息 |



## 使用示例

### 基本使用
```cmd
colorTree -p "%localappdata%/User Data"
```

### 显示3层深度
```cmd
colorTree --Layer 3
```

### 组合使用选项
```cmd
colorTree -s 3 -of 2 -a 100 -fk -l 4 -od
```

## 配色方案说明
支持0-8共9种预设配色方案，每个方案包含7种颜色循环使用。当起始偏移量超过6时自动取余（如8→1）。

## 单位说明
- 默认使用二进制单位（KiB/MiB/GiB）
- 启用`-k`选项后使用十进制单位（KB/MB/GB）进行计算（使用1000进制）
- 使用`-fk`选项则强制以十进制单位（KB/MB/GB）显示（不影响进制）
